<br>
<p style="margin-left: 20px;"> Clock shows <b>h</b> hours, <b>m</b> minutes and <b>s</b> seconds after midnight.<br>
    Your task is to write a function which returns the time since midnight in milliseconds.</p>
<h4>Example:</h4>
<p>h = 0<br>
m = 1<br>
s = 1<br>
<br>
result = 61000</p>
